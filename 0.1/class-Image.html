Image

An instance of Image represents an image to be displayed on a Page or Key.

Example
const { openStreamDeck } = require('streamdeck-ui-node');

(async () => {
  const streamDeck = await openStreamDeck();
  const page = streamDeck.createPage();
  const key = streamDeck.createKey();

  let pageBackgroundImage = streamDeck.createPageBackgroundImage('path/to/image.png');
  pageBackgroundImage = page.createBackgroundImage('path/to/image.png');

  let keyBackgroundImage = streamDeck.createKeyBackgroundImage('path/to/image.png');
  keyBackgroundImage = key.createBackgroundImage('path/to/image.png');

  let keyImage = streamDeck.createKeyImage('path/to/image.png');
  keyImage = key.createImage('path/to/image.png');
})();

Type Definitions

Source

An image Source can be any of the following;

  • An image file path string to be loaded by sharp - constructor.
  • An image Buffer to be loaded by sharp - constructor.
  • An existing Image instance.
  • An Array of Sources.
  • An Object with the following properties, requiring at least one of the first three properties;
Properties
NameTypeDefaultDescription
[source]string |
Buffer |
Image

One of the first three Source types listed above.

[empty]boolean

A true value will create a completely transparent image source. The source property takes priority.

[color]Color

A Color value will create a solid Color image source. The source and empty properties take priority.

[sharpOptions]Object

Options to be passed to sharp - constructor when loading the source. Does not apply if the source is an Image.

[resize]booleantrue

Whether or not the Source should be resized before the compositing step of the image load process.

[resizeOptions]Object

Options to be passed to sharp - resize during the first resizing step of the image load process.

[compositeOptions]Object

Options to be passed to sharp - composite during the compositing step of the image load process.

Color

A Color can be any of the following;

  • A color name string recognized by the colornames library.
  • An rgb(a) hexadecimal color string, with or without a leading #. (e.g. 0f0, 0f0f, #00ff00, #00ff00ff)
  • An Array of numbers with a length of either 3 or 4, each value being between 0 and 255 inclusive.
  • An Object with the following color channel properties;
