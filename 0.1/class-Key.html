Key

An instance of Key represents a virtual key on a Page created via StreamDeck.createKey.

Example
const { openStreamDeck } = require('streamdeck-ui-node');

(async () => {
  const streamDeck = await openStreamDeck();

  const key = streamDeck.createKey();
})();

constant KEY_COUNT number

The number of keys on the Stream Deck.

constant KEY_WIDTH number

The width of each key on the Stream Deck in pixels.

constant KEY_HEIGHT number

The height of each key on the Stream Deck in pixels.

readonly HOLD_TIME number|undefined

The number of milliseconds after the down event is triggered that the hold event will be triggered. A value of 0 means the hold event is disabled. If undefined Page#hold events will be allowed to propagate from Pages that the Key is attached to.

readonly PRESS_TIME number

The number of milliseconds after the down event is triggered that the press scaling effect will apply. A value of 0 means the press scaling effect lasts until the up event is triggered.

readonly PRESS_SCALE number

The default scale the Key will use for the press scaling effect. A value of 1 means the press scaling effect is disabled.

readonly IDLE_TIME number

The number of milliseconds of inactivity before the idle event is triggered. A value of 0 means the idle event is disabled.

readonly backgroundImage Image|undefined

The background image of the Key.

readonly image Image|undefined

The image of the Key.

readonly destroyed boolean

Whether or not the Key has been destroyed.
