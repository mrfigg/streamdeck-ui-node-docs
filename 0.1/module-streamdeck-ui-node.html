Type Definitions

StreamDeckInfo

An Object describing a connected Stream Deck device.

Properties
NameTypeDescription
hidPathstring

The HID path to the Stream Deck device.

modelStreamDeckModel

A short string describing the Stream Deck model.

serialNumberstring

The serial number found on the Stream Deck device.

StreamDeckModel

A short string describing a Stream Deck model.

Can be one of: 'original', 'originalv2', 'original-mk2', 'mini', 'miniv2', 'xl', 'xlv2', 'pedal', 'plus'

StreamDeckOptions

A Object holding options for creating a StreamDeck instance.

Properties
NameTypeDefaultDescription
[holdTime]number500

The number of milliseconds after the StreamDeck#down event is triggered that the StreamDeck#hold event will be triggered. Setting to 0 disables the StreamDeck#hold event.

(Integer greater than or equal to 0.)

[pressTime]number0

The default number of milliseconds after the Key#down event is triggered that the press scaling effect will apply on Keys created by the StreamDeck. Setting to 0 makes the press scaling effect last until the Key#up event is triggered.

(Integer greater than or equal to 0.)

[pressScale]number0.85

The default scale any Keys created by the StreamDeck will use for the press scaling effect. Setting to 1 disables the press scaling effect.

(Float between 0 and 2 inclusive.)

[idleTime]number10000

The number of milliseconds of inactivity before the StreamDeck#idle event is triggered. Setting to 0 disabled the StreamDeck#idle event.

(Integer greater than or equal to 0.)

[brightness]number1

The initial brightness of the StreamDeck's panel.

(Float between 0 and 1 inclusive.)

[on{Event}]function

An event listener that will be attached to the StreamDeck on {event}.

See StreamDeck for a list of available events.

[{customProperty}]*

Any custom properties not already reserved by the StreamDeck class will be added to the StreamDeck.
