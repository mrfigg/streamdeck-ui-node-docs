StreamDeck

An instance of StreamDeck represents a Stream Deck device accessed via streamdeck-ui-node.openStreamDeck or streamdeck-ui-node.manageStreamDeck.

Example
const { openStreamDeck } = require('streamdeck-ui-node');

(async () => {
  const streamDeck = await openStreamDeck();
})();

The number of milliseconds after the down event is triggered that the hold event will be triggered. A value of 0 means the hold event is disabled.

The default number of milliseconds after the Key#down event is triggered that the press scaling effect will apply on Keys created by the StreamDeck. A value of 0 means the press scaling effect last until the Key#up event is triggered.

The default scale any Keys created by the StreamDeck will use for the press scaling effect. A value of 1 means the press scaling effect is disabled.

The number of milliseconds of inactivity before the idle event is triggered. A value of 0 means the idle event is disabled.

destroy()

Clean up all internal state data used by the StreamDeck in preparation for garbage collection. This also destroys all Pages and Keys created by the StreamDeck.

close() → Promise

Destroy the StreamDeck and all Pages and Keys created by the StreamDeck, then close STREAMDECK_NODE.

Returns a Promise that resolves once STREAMDECK_NODE has been closed.
